# About

These tools are built for the [XRRPG](https://publish.obsidian.md/xeonrapture/xr/XRRPG+2.0/XRRPG+2.0+Core+Rules), a sci-fantasy TTRPG built for flashy, narrative-focused anime fights where you manipulate the laws of physics.

All this is part of the [Xeon Rapture](https://publish.obsidian.md/xeonrapture/xr/Xeon+Rapture) project, a work of sci-fantasy collaborative storytelling.
